# ✈️ Chatbot para Agência de Turismo

Projeto acadêmico desenvolvido para a disciplina de **Inteligência Artificial**, utilizando **HTML, CSS e JavaScript**.

O sistema simula um chatbot de atendimento para uma agência de turismo, realizando perguntas ao usuário, validando respostas através de palavras-chave e exibindo um resumo final das escolhas da viagem.

## 🚀 Funcionalidades

- Chat interativo em tempo real
- Perguntas sequenciais sobre a viagem
- Sistema simples baseado em conhecimento
- Validação de respostas utilizando palavras-chave
- Exibição das escolhas realizadas pelo usuário
- Popup de confirmação com SweetAlert2
- Interface organizada e responsiva

---

## 📋 Perguntas do Chatbot

O sistema realiza perguntas relacionadas à viagem, como:

- País de destino
- Quantidade de passageiros
- Tipo de assento
- Dia da viagem
- Interesse em pacotes turísticos

As respostas válidas são armazenadas e apresentadas ao final do atendimento.
