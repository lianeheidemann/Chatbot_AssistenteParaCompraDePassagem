# ✈️ Agência de Turismo

Projeto acadêmico desenvolvido utilizando **HTML, CSS e JavaScript**, simulando um sistema simples de atendimento para uma agência de turismo em formato de chatbot.

O sistema realiza perguntas ao usuário, valida respostas com base em palavras-chave pré-definidas e exibe um resumo final das escolhas da viagem.

## 🚀 Funcionalidades

- Chat interativo com perguntas sequenciais
- Sistema básico baseado em conhecimento
- Validação de respostas utilizando palavras-chave
- Resumo final das opções escolhidas
- Popup de confirmação com SweetAlert2
- Interface simples e responsiva

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- SweetAlert2

---

## 📋 Perguntas do Sistema

O chatbot realiza perguntas como:

- País de destino
- Quantidade de passageiros
- Tipo de assento
- Dia da viagem
- Interesse em pacotes turísticos

---

## 🧠 Lógica do Projeto

O sistema funciona através de:

- Arrays de perguntas
- Arrays de palavras-chave válidas
- Controle de fluxo usando JavaScript
- Atualização dinâmica do DOM

As respostas válidas são armazenadas e exibidas ao final do atendimento.
